## Classification & Flag Legend

- Classes
  - Promote: no grammar violation AND no stdout violation
  - Non-Conformant: grammar violation OR stdout violation

  - `[G]`: Grammar
  - `[S]`: Stdout
  - `[E]`: Exit Code
  - `[U]`: Unknown/Unverified
- Evidence
  - `Doc`
  - `Source`
  - `Observed`
  - `Provisional`
- Intent
  - `Operator`
  - `Developer`
  - `Experimental`
